# Aplicação de Clima usando vanilla Javascript
Esse é um projeto recomendado para quem está iniciando em Javascript. Essa aplicação utiliza da API [Weather Api](https://www.weatherapi.com) para obter informações de temperatura e clima.

## Pontos Treinados
* Manipulação de DOM
* Uso de Fetch, Await e Async
* Função setInterval (para atualizar o horário constantemente)
* Estruturação de HTML e estilização básica com CSS

